## Usage

### Mouse/Touch Input
- Click on any button to perform calculations
- Use the display to view current input and results

### Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `0-9` | Number input |
| `+` | Addition |
| `-` | Subtraction |
| `*` | Multiplication |
| `/` | Division |
| `.` | Decimal point |
| `Enter` or `=` | Calculate result |
| `Backspace` | Remove last digit |
| `Escape` or `C` | Clear all |
| `%` | Percentage |
